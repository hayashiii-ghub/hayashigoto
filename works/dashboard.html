title:図面記号検出システム
year:2025
role:Full Stack
stack:Claude Code Neon Gemini API

Gemini 3 ThinkingのAPIを活用し、図面画像の中から特定の記号を自動で検出・抽出する社内システムを開発しました。

図面をアップロードするとAIが記号の位置と種類を解析し、結果をNeonのデータベースに蓄積。過去の検出履歴の検索や集計にも対応しています。

開発にはClaude Codeを全面的に活用し、プロンプト設計からAPI連携、データベース設計、UIの実装までを高速に構築しました。
